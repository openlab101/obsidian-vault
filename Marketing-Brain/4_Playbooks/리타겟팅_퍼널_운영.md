# 리타겟팅 퍼널 운영 플레이북

> 순차적 리타겟팅 퍼널(인지→고려→전환)을 설계하고 운영하는 실전 가이드

---

## 퍼널 구조 요약

```
[브로드 타겟] → 인지 광고
        ↓ (영상 50%+ / 클릭 / 프로필 방문)
[맞춤 오디언스 A] → 고려 광고
        ↓ (영상 75%+ / 클릭 / 상세페이지 방문)
[맞춤 오디언스 B] → 전환 광고 (프로모션/할인코드)
```

| 퍼널 | 목적 | 소재 유형 | KPI | 리타겟팅 조건 |
|---|---|---|---|---|
| 인지 | 브랜드 인지도 확대, 신규 잠재고객 유입 | 행사 현장 분위기 사진, 시식 후기 영상 | CPM / 도달율 / VTR | - |
| 고려 | 제품 관심 및 구매 고려 유도 | 고객 인터뷰/후기 영상, 제품 상세 컷 | CTR / CPC / 상세페이지 체류시간 | 인지 영상 50%+ 재생 + 클릭 + 프로필 방문자 |
| 전환 | 구매 전환 / 재구매 유도 | 한정 프로모션 소재, 할인 코드 배포 | CPA / ROAS / 구매 전환율 | 고려 영상 75%+ 재생 + 클릭 + 상세페이지 방문자 |

---

## Phase 1: 사전 준비

### 필수 세팅
- [ ] Meta 픽셀 설치 확인 (상세페이지 + 구매 완료 페이지)
- [ ] 번들 상품 랜딩페이지 준비 (단품은 배송비 포함 시 적자 → [[제품별_마진율]] 참고)
- [ ] 소재 최소 2~3개씩 준비 (A/B 테스트용)

### 맞춤 오디언스 사전 생성 (광고 시작 전에 반드시)
경로: 광고 관리자 → ≡ → 타겟 → 맞춤 타겟 만들기

| 오디언스 | 설정 | 기간 |
|---|---|---|
| 영상 50% 시청자 | 맞춤 타겟 → 동영상 → 50% 이상 시청 | 30일 |
| 영상 75% 시청자 | 맞춤 타겟 → 동영상 → 75% 이상 시청 | 30일 |
| 광고 클릭자 | 맞춤 타겟 → Meta 소스 → Instagram → 광고 참여자 | 30일 |
| 프로필 방문자 | 맞춤 타겟 → Instagram 프로필 방문자 | 30일 |
| 상세페이지 방문자 | 맞춤 타겟 → 웹사이트 → 특정 URL | 30일 |

> 소급 적용 안 됨. 반드시 광고 시작 전에 생성해 둘 것.

## Phase 2: 예산 배분 (100만원 / 4주 기준)

### 주간 예산

| 주차 | 인지 | 고려 | 전환 | 합계 |
|---|---|---|---|---|
| 1주차 | 20만원 | - | - | 20만원 |
| 2주차 | 15만원 | 10만원 | - | 25만원 |
| 3주차 | 10만원 | 10만원 | 10만원 | 30만원 |
| 4주차 | 5만원 | 5만원 | 15만원 | 25만원 |
| **합계** | **50만원** | **25만원** | **25만원** | **100만원** |

> 1~2주차에 오디언스 쌓고, 3~4주차에 전환 집중. 처음부터 3단계 동시 운영 금지.

→ 더 큰 예산은 [[예산배분_의사결정]] 참고

## Phase 3: KPI 기준값

### 인지 단계

| KPI | 목표 | 🟢 정상 | 🟡 관찰 | 🔴 조치 |
|---|---|---|---|---|
| CPM | 5,000~10,000원 | ≤ 8,000 | 8,000~12,000 | > 12,000 |
| VTR (3초) | 30% 이상 | ≥ 35% | 25~35% | < 25% |
| 50% 재생률 | 10~15% | ≥ 15% | 10~15% | < 10% |

### 고려 단계

| KPI | 목표 | 🟢 정상 | 🟡 관찰 | 🔴 조치 |
|---|---|---|---|---|
| CTR | 1.5% 이상 | ≥ 2% | 1~1.5% | < 1% |
| CPC | 300~500원 | ≤ 300원 | 300~500원 | > 500원 |
| 상세페이지 체류 | 30초 이상 | ≥ 40초 | 20~30초 | < 15초 |
| 이탈률 | 70% 이하 | ≤ 60% | 60~70% | > 70% |

### 전환 단계

| KPI | 목표 | 🟢 정상 | 🟡 관찰 | 🔴 조치 |
|---|---|---|---|---|
| CPA | ≤ 3,000원 | ≤ 2,000원 | 2,000~3,000원 | > 3,000원 |
| ROAS | ≥ 1.0 | ≥ 1.5 | 1.0~1.5 | < 1.0 |
| 구매 전환율 | 2~3% | ≥ 3% | 1~2% | < 1% |
| 장바구니 전환율 | 5~8% | ≥ 8% | 3~5% | < 3% |

## Phase 4: 리타겟팅 풀 모니터링

### 확인 경로
광고 관리자 → 타겟 → 해당 오디언스 클릭 → 크기 확인

### 주간 목표

| 주차 | 확인 항목 | 목표치 |
|---|---|---|
| 1주차 말 | 영상 50% 시청자 | 500명+ |
| 2주차 말 | 영상 50% 시청자 | 1,000명+ (고려 광고 집행 가능) |
| | 광고 클릭자 | 300명+ |
| 3주차 말 | 상세페이지 방문자 | 500명+ |
| 4주차 말 | 전체 리타겟팅 풀 합산 | 2,000명+ |

> Meta 최소 오디언스: 1,000명. 이 이하면 리타겟팅 집행 불가.

### 간접 확인법
광고 관리자 보고서 → 열 맞춤 설정 → "동영상 50% 재생" 추가 → 이 숫자 ≈ 영상 시청자 오디언스 크기

---

## Phase 5: 소재 효율 확인 & ROAS 개선

### 소재 등급 분류 (매주 월요일)

**인지 소재:**

| 등급 | VTR (3초) | 50% 재생률 | CPM | 판단 |
|---|---|---|---|---|
| A | 35%+ | 15%+ | ≤ 8,000원 | 예산 증액 |
| B | 25~35% | 10~15% | 8,000~12,000원 | 유지 관찰 |
| C | < 25% | < 10% | > 12,000원 | OFF |

**고려 소재:**

| 등급 | CTR | CPC | 판단 |
|---|---|---|---|
| A | 2%+ | ≤ 300원 | 예산 증액 |
| B | 1~2% | 300~500원 | 유지 관찰 |
| C | < 1% | > 500원 | OFF |

**전환 소재:**

| 등급 | CPA | ROAS | 판단 |
|---|---|---|---|
| A | ≤ 2,000원 | 1.5+ | 예산 집중 |
| B | 2,000~3,000원 | 1.0~1.5 | 유지, 소재 변형 테스트 |
| C | > 3,000원 | < 1.0 | OFF |

### ROAS 개선 사이클

```
C등급 소재 OFF
     ↓
절약된 예산을 A등급 소재에 재배분
     ↓
A등급 소재의 변형 버전 1~2개 추가 테스트
     ↓
매주 반복
```

### A등급 소재 분석 포인트

| 항목 | 질문 | 반영 |
|---|---|---|
| 첫 1~2초 | 어떤 장면으로 시작했나? | 다음 소재도 같은 훅 패턴 |
| 포맷 | 리얼스? 이미지? | 잘 되는 포맷 위주 제작 |
| 메시지 | "저당"? "편의성"? | 잘 되는 소구점 중심 카피 |
| 타겟 | 어떤 연령/성별 반응? | 타겟 세분화 반영 |

→ 소재 성과 기록은 [[메타_소재별_성과누적]] 참고

### 주간 점검 루틴 (15분)

1. 광고 관리자 → 광고 탭 열기 (2분)
2. 지난 7일 기준 정렬 (1분)
3. 소재별 A/B/C 분류 (5분)
4. C등급 OFF (1분)
5. A등급 예산 증액 (2분)
6. A등급 변형 소재 아이디어 메모 (4분)

---

## 주의사항

- 첫 달은 ROAS 적자가 정상 → 데이터 수집이 목표
- 소재 판단은 최소 1,000 노출 또는 3일 이후
- A등급 소재도 2주 후 효율 하락 (소재 피로도) → 변형 소재로 교체
- 전환 목표는 반드시 **번들** (단품은 배송비 포함 시 적자)
- 오디언스 빈도(frequency) 3 이상이면 소재 피로도 의심
