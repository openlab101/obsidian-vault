# 2. Areas

## 폴더 목적

마감일 없이 지속적으로 관리해야 하는 책임 영역과 관심사를 다루는 공간입니다.

## 들어가는 노트

- 지속적인 관리가 필요한 영역 (건강, 재무, 커리어 등)
- 반복적으로 수행하는 업무
- 표준 프로세스나 체크리스트
- 예시: CRM 운영, 데이터 분석, 그로스 마케팅 전략

## 작업 방식과 규칙

1. **영역 정의**: 각 Area는 명확한 책임 범위를 가짐
2. **지속성**: 완료되지 않고 계속 관리되는 주제
3. **정기 리뷰**: 주기적으로 내용을 검토하고 업데이트
4. **프로젝트 연결**: Area에서 파생된 프로젝트는 `1. Projects/`에 생성

## Area 노트 구조 예시

```markdown
# [영역명]

## 목표/목적
이 영역에서 달성하고자 하는 것

## 주요 활동
- 활동 1
- 활동 2

## 관련 프로젝트
- [[Project 1]]
- [[Project 2]]

## 참고 자료
- [[Resource]]
```
