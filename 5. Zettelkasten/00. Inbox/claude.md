# 00. Inbox

## 폴더 목적

빠른 메모와 처리되지 않은 생각을 임시로 담아두는 받은편지함입니다.

## 들어가는 노트

- 갑자기 떠오른 아이디어
- 나중에 정리할 메모
- 임시로 저장한 정보
- 처리가 필요한 항목들

## 작업 방식과 규칙

1. **빠른 캡처**: 생각이 떠오르면 즉시 기록, 완벽함보다 속도 우선
2. **정기 처리**: 매일 또는 주간 단위로 Inbox 비우기
3. **분류 이동**:
   - 발전시킬 가치가 있는 아이디어 → `20. Permanent/`
   - 읽은 자료 요약 → `10. Literature/`
   - 실행할 항목 → `1. Projects/` 또는 `2. Areas/`
   - 참고 자료 → `3. Resources/`
4. **삭제**: 가치 없는 메모는 과감히 삭제

## Inbox 노트 작성 예시

```markdown
# 메모 제목

[빠르게 기록한 내용]

## 다음 단계
- [ ] Literature 노트로 발전시키기
- [ ] 관련 영구 노트 찾기
- [ ] 프로젝트로 전환 검토
```

## 목표

**Inbox를 비워두는 것이 목표**입니다. 항목이 쌓이지 않도록 정기적으로 처리하세요.
