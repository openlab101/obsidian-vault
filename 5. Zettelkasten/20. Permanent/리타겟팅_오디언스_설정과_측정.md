# 리타겟팅 오디언스 설정과 측정

## 핵심 아이디어
Meta 광고에서 리타겟팅은 "맞춤 오디언스"를 통해 작동한다. 광고에 반응한 사용자를 자동으로 모아주는 오디언스를 사전에 만들어 두고, 크기가 1,000명 이상 쌓이면 다음 퍼널 단계의 타겟으로 사용한다.

## 설명

### 오디언스 생성 (광고 시작 전 필수)
경로: 광고 관리자 → ≡ → 타겟 → 맞춤 타겟 만들기

| 오디언스 | 설정 |
|---|---|
| 영상 50% 시청자 | 맞춤 타겟 → 동영상 → 50% 이상 시청한 사람 |
| 영상 75% 시청자 | 맞춤 타겟 → 동영상 → 75% 이상 시청한 사람 |
| 광고 클릭자 | 맞춤 타겟 → Meta 소스 → Instagram → 광고 참여자 (클릭) |
| 프로필 방문자 | 맞춤 타겟 → Instagram 프로필 방문자 |
| 상세페이지 방문자 | 맞춤 타겟 → 웹사이트 → 특정 URL (픽셀 필요) |

**중요**: 소급 적용이 안 된다. 광고 돌린 후에 만들면 이전 데이터는 수집되지 않음.

### 크기 확인 방법
경로: 광고 관리자 → 타겟 → 해당 오디언스 클릭 → "크기" 확인

- 생성 후 반영까지 24~48시간 소요
- "1,000명 미만"으로 표시되면 집행 불가
- 수시간 단위로 갱신 (실시간 아님)

### 간접 확인법
광고 관리자 보고서 → 열 맞춤 설정 → "동영상 50% 재생" 열 추가
→ 이 숫자 ≈ 영상 시청자 오디언스 크기

### Meta 영상 재생률 구간
Meta는 25% / 50% / 75% / 95% 단위로만 오디언스 생성 가능. "55% 이상 재생" 같은 커스텀 값은 설정할 수 없으므로, 가장 가까운 상위 구간(75%)을 사용.

## 예시
바이트 인지 광고 1주차 보고서:
- 도달: 12,000명
- 3초 재생: 3,600회 (VTR 30%)
- 50% 재생: 1,400회 → 리타겟팅 풀 약 1,400명 (집행 가능)
- 링크 클릭: 180회

→ 2주차부터 고려 광고 타겟으로 사용 가능

## 내 적용
오디언스는 광고 세팅할 때 가장 먼저 만들어 둔다. 매주 타겟 페이지에서 크기 확인하여 1,000명 넘으면 다음 퍼널 오픈. 빈도(frequency) 3 이상이면 오디언스가 소진되고 있다는 신호.
