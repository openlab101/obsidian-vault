# 광고소재 ABC 등급분류

## 핵심 아이디어
광고 소재를 A/B/C 3등급으로 분류하여, C는 끄고 A에 예산을 집중하고 A의 변형을 테스트하는 사이클을 반복하면 같은 총 예산으로 ROAS를 개선할 수 있다.

## 설명

### 분류 기준 (퍼널별)

**인지 소재:**
- A등급: VTR 35%+, 50% 재생률 15%+, CPM ≤ 8,000원 → 예산 증액
- B등급: VTR 25~35%, 50% 재생률 10~15% → 유지 관찰
- C등급: VTR < 25%, 50% 재생률 < 10% → OFF

**고려 소재:**
- A등급: CTR 2%+, CPC ≤ 300원 → 예산 증액
- B등급: CTR 1~2%, CPC 300~500원 → 유지 관찰
- C등급: CTR < 1%, CPC > 500원 → OFF

**전환 소재:**
- A등급: CPA ≤ 2,000원, ROAS 1.5+ → 예산 집중
- B등급: CPA 2,000~3,000원, ROAS 1.0~1.5 → 변형 테스트
- C등급: CPA > 3,000원, ROAS < 1.0 → OFF

### ROAS 개선 사이클

```
C등급 OFF → 절약 예산을 A등급에 재배분 → A등급 변형 소재 추가 → 매주 반복
```

### A등급 소재에서 뽑아낼 인사이트
- **훅(첫 1~2초)**: 어떤 장면이 시선을 잡았나
- **포맷**: 리얼스 vs 이미지 vs 카드뉴스
- **메시지**: 어떤 소구점("저당", "편의성", "맛")이 먹혔나
- **타겟**: 어떤 연령/성별에서 반응이 좋았나

이 요소를 다음 소재 기획에 반영하면 승률이 올라간다.

## 예시
소재 3개 운영 중 1주일 후:
- 소재A (시식 후기 영상): VTR 38%, CTR 2.1% → A등급, 예산 +50%
- 소재B (제품 상세 컷): VTR 28%, CTR 1.3% → B등급, 유지
- 소재C (행사 사진): VTR 18%, CTR 0.6% → C등급, OFF

→ 소재A의 "리얼 시식 반응" 훅이 효과적 → 같은 형식으로 변형 소재 제작

## 내 적용
매주 월요일 15분 루틴: 광고 탭 → 7일 기준 정렬 → A/B/C 분류 → C OFF → A 증액 → 변형 아이디어 메모. 소재 판단은 최소 1,000 노출 또는 3일 이후에. A등급 소재도 약 2주 후 효율이 떨어지므로(소재 피로도) 변형 소재를 미리 준비해야 한다.
