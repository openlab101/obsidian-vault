# 3. Resources

## 폴더 목적

관심 있는 주제에 대한 참고 자료와 외부 정보를 정리하는 저장소입니다.

## 들어가는 노트

- 북마크와 웹 링크 모음
- 강의 자료 및 코스 노트
- 도구 및 템플릿 모음
- 가이드 문서 및 치트시트
- 예시: 마케팅 도구 모음, 디자인 레퍼런스, 데이터 분석 가이드

## 작업 방식과 규칙

1. **주제별 분류**: 관련 주제끼리 폴더나 태그로 그룹화
2. **참고 전용**: 직접 작업하지 않고 참고만 하는 자료
3. **출처 기록**: 외부 자료는 출처 URL이나 저자 정보 포함
4. **활용 연결**: Projects나 Areas에서 필요시 링크로 참조

## Resource 노트 구조 예시

```markdown
# [자료명]

## 출처
[URL 또는 저자]

## 요약
간단한 설명

## 주요 내용
- 핵심 포인트 1
- 핵심 포인트 2

## 활용
이 자료를 어디에 활용할 수 있는지
```
